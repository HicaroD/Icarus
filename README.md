## Installation

1. [Plug][]

    In your `.vimrc` or `init.vim` (For NeoVim):
    ```
    Plug 'HicaroD/Icarus'
    ```

2. [Vundle][]

    In your `.vimrc` or `init.vim` (For NeoVim):
    ```
    Plugin 'HicaroD/Icarus'
    ```

[vundle]: https://github.com/gmarik/Vundle.vim
[plug]: https://github.com/junegunn/vim-plug

## Usage

Add this to your `.vimrc`:

```
set termguicolors
colorscheme icarus
```
